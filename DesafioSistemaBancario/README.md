<h1>Desafio DIO - Criando um sistema bancário com funções Python</h1>

<h2>Objetivo Geral</h2>

<p>Criar um sistema básico em python que tenha funções bancárias padrões, por exemplo: saque, depósito, extrato, cadastrar usuário, e cadastrar conta bancária.
</p>

<h2>Desafio</h2>

<p>Tratamos nosso código de maneira modular, criamos as seguintes funções: Menu, depositar, sacar, exibir extrato, criar usuário (cliente do banco), criar conta (vincular com usuário), filtrar usuário, listar contas e visualizar histórico.
</p>

<h2>Separação em funções</h2>

<p>Criamos funções para todas as operações em nosso sistema. Para exercitar tudo o que foi aprendido no módulo do bootcamp da DIO. Cada função tem uma regra na passagem de argumentos. O retorno e a forma como são chamadas, são melhor definidas dentro do código.
</p>

<h2>Criar usuário (cliente)</h2>

<p>Armazenamos os usuários em uma lista, um usuário é composto por: nome, data de nascimento, cpf e endereço. O endereço é uma string com o formato: logradouro, nro - bairro - cidade/sigla estado. Só podemos armazenar os números do CPF e não podemos cadastrar 2 usuários com o mesmo CPF.

<h3>Criar conta corrente</h3>

<p>Também armazenamos contas em uma lista, uma conta é composta por: agência, número da conta e usuário. O número da conta é sequencial, iniciando em 1. O número da agência é fixo: "0001". O usúario pode ter mais de uma conta, mas uma conta pertence a somente um usuário.
</p>

<h2>Resolução</h2>

<p>Neste módulo do bootcamp fui capaz de aprender a dominar funções python, tipos de operadores, estruturas condicionais e de repetição, a manipular strings e dicionários com multiplas listas.
O aprendizado adquirido até este ponto do módulo me deu confiança e capacidade para introduzir-me ao universo python e dizer que, me sinto promissor em trilhar esta jornada de aprendizado profundo junto com a DIO.  
</p>
